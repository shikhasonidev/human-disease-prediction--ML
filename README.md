# Human Disease Prediction using Machine Learning

## Overview
This project predicts human diseases using machine learning based on patient health data.

## Features
- Data preprocessing
- Random Forest model
- Evaluation metrics

## Dataset
- Age
- Sex
- Glucose
- BMI
- Blood Pressure
- Cholesterol

## Results
- Accuracy: ~94%

## Run
pip install -r requirements.txt
python main.py
